# Live Tokens — Releases

Downloads and auto-update feed for the **Live Tokens** macOS app — a menu-bar + dashboard app for live Claude/Codex token usage and cost.

- **Installing:** grab the `.dmg` from the [latest release](https://github.com/eiselmayer/live-tokens-releases/releases/latest).
- The `.zip` and `latest-mac.yml` assets are the auto-update feed consumed by the app itself — no need to download them manually.

The app's source code lives in a private repository.
